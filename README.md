# TimeTracker Bot

A Telegram bot for tracking time spent on tasks with charts and statistics.

## Features
- Start and end tasks with buttons
- Timer for active task  
- Statistics: today, week, month
- Pie chart visualization
- SQLite database storage

## Technologies
- Python 3
- Aiogram 3 (Telegram API)
- SQLite3
- Matplotlib (charts)

1. Send `/start` to your bot
